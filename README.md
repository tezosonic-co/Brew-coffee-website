
## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Enjoy the smooth coffee shop experience!

## 📱 Responsive Breakpoints

- Mobile: < 768px
- Tablet: 768px – 1024px
- Desktop: > 1024px

## ✨ Skills Demonstrated

- Flexbox & CSS Grid
- Responsive navbar implementation
- CSS hover animations & transitions
- Card component design
- Media queries for adaptability
- Clean UI/UX principles
